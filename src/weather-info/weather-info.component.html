About the Weather Data

Data Source

Weather data is provided by Open-Meteo, a free and open-source weather API that aggregates data from multiple weather models.

Update Frequency

Weather data is updated in real-time with:

  • Current conditions: Updated every hour
  • Forecasts: Updated 4 times daily
  • Historical data: Updated daily

Data Coverage

The weather data includes:

  • Temperature and "feels like" temperature
  • Precipitation probability
  • Sunrise and sunset times
  • Hourly forecasts for the current day
  • Daily forecasts up to 16 days ahead
